    Help & Support

    Frequently Asked

    Is SmartShare completely free?
    Yes. SmartShare requires no accounts, subscriptions, or logins. You remain completely anonymous while using the service.
    What is the maximum file size?
    Direct (P2P): Unlimited. Files transfer via your local network or WebRTC tunnel, bypassing our servers.

    Link Share (Cloud): 10MB per file (100MB for video files) to maintain fast cloud infrastructure.
    Are my files encrypted?
    Direct Transfers use DTLS 1.3 encryption, creating a secure, end-to-end encrypted tunnel. We cannot see or store your files.

    Link Shares are transmitted via HTTPS and stored in an isolated vault that automatically permanently deletes files on expiration.
    Why is Direct Transfer stuck on "Connecting..."?
    Strict corporate firewalls, VPNs, or Symmetric NAT networks can block WebRTC P2P traffic.

    The Fix: Switch one device to Mobile Data, or use the Link Share mode instead.
